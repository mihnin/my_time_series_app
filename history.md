# История изменений

## 2023-03-16
### Улучшения системы управления и администрирования
- **Административная панель**:
  - Создана отдельная страница администрирования с защитой паролем (admin/admin)
  - Добавлены вкладки для управления системными настройками
  - Реализован мониторинг ресурсов системы в реальном времени
  - Добавлено управление задачами в очереди с возможностью отмены
  - Реализовано управление сессиями пользователей
  - Добавлен просмотр и фильтрация логов системы

- **Автоматическое определение параметров**:
  - Создан модуль `src/data/auto_detect.py` для интеллектуального определения полей данных
  - Добавлена функция определения колонки с датой на основе имени и типа данных
  - Добавлена функция определения колонки ID на основе категориальных признаков
  - Добавлена функция определения целевой колонки среди числовых полей
  - Реализовано автоматическое определение частоты временного ряда (день, неделя, месяц)
  - Все настройки автоопределения вынесены в конфигурационный файл

- **Исправления ошибок**:
  - Увеличен порог использования памяти в конфигурации (с 90% до 95%) для предотвращения предупреждений о перегрузке
  - Исправлена ошибка с парсингом дат (убран параметр dayfirst=True)
  - Улучшена обработка ошибок при загрузке данных
  - Улучшено сохранение состояния между сессиями

- **Улучшения UI**:
  - Добавлена кнопка для автоопределения полей
  - Добавлена кнопка для автоопределения частоты временного ряда
  - Выполнена визуализация статуса ресурсов системы
  - Добавлено отображение метаданных модели на главной странице

- **Улучшения конфигурации**:
  - Все технические параметры вынесены в раздел конфигурации
  - Добавлены настройки шаблонов для автоопределения полей
  - Добавлены настройки форматов дат для улучшения распознавания
  - Добавлены настройки пороговых значений для определения частоты
